## Source   
Оваа верзија вклучува `автоматско семплирање на центроиди`, исто така во претходните верзии иницијалните центроиди се бришеа од датасетот, што создаваше **наклонетост** и ставен е `праг на конвергенција`. Максималниот број на итерации не се задава од корисникот, туку е кодиран во алгоритмот. Поставен е на `максимум 100 итерации`.
