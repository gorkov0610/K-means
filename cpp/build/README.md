## Build   
Постигната е последната верзија за програмата. 
За влез се користи JSON објект, во него се специфицира **датасет**, **полиња од датасетот**, **број на кластери** и **центроиди** (*опционално*). Влезот изгледа вака:        
  
>{  
>    "dataset":"path/to/dataset",     
>    "fields":["names", "of", "fields"],        
>    "numClusters":2,  
>    "centers":[{"x":0.15, "y":0.1, "z":12.2}, {"x":9.5, "y":1.7, "z":1.1}]  
>}   

На излез од програмот повторно се добива JSON објект во кој се содржани **Калински-Харабаш индекс**, **центроидите на кластерите** и **кластерите**, сите полиња се низа од низи. Погледни го `output.json`.
